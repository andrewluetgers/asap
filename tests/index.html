asap "tests"

Not quite unit tests but this set of examples and comparisons provide some evidence of functionality. Pull up your browsers console to see more verbose logging.

simple comparative examples

simple.html
A simple baseline that loads jQuer, jQuery UI and simple.js at the bottom of the page with no prelaoding solution employed.
simple-asap.html
The baseline example loading js with asap. Asap is used again in the onload handler to lazy load a jQuery plugin and run it once loaded using the codeReady callback.
simple-lab.html
The baseline example loading js with LABjs.
simple-require.html
The baseline example loading js with requireJS.
